# Uno Bot - Java
Discord bot that plays Uno in servers.
## Installation
1. Sign in to Discord in your browser of choice.
2. Click this link [here](https://discordapp.com/api/oauth2/authorize?client_id=436627620705075200&permissions=329728&redirect_uri=https%3A%2F%2Fdiscordapp.com%2Fapi%2Foauth2%2Fauthorize%2F&response_type=code&scope=bot%20messages.read "Uno Bot Invite Link").
3. Play Uno in your discord server!
## Commands
* !help - Provides all the commands in the form of a discord direct message.
* !rules - Direct messages the user Uno rules to whomever sends !rules.
* !start - This can do one of two things.
    - If a game isn't already in progress in your server, it queues up a gamefor you and your friends!
    - If a game is in progress, and you already have the number of players you want, using this command again will start the game.
* !join - If game creation is in progress, you will join that game.
* !play [cardNumber] - Plays a card from your hand numbered starting at 1 left to right.
* !draw - If it's your turn, draws a card.
